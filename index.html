Добро пожаловать

Это демонстрация веб-интерфейса, созданного с учётом потребностей людей с различными ограничениями.

Подробнее о доступности веб-приложений

Доступность веб-приложений для людей с ограничениями по здоровью означает создание интерфейсов, которые удобны и понятны для всех пользователей, независимо от их физических или когнитивных особенностей. Это включает поддержку экранных дикторов, навигацию с клавиатуры, достаточный контраст текста, масштабируемость интерфейса, субтитры и простую структуру. Цель доступности — обеспечить равные условия использования цифровых продуктов для всех категорий граждан.
Вы можете использовать клавишу Tab для навигации.
